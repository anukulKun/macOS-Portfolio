# About This Site

This portfolio is built to mimic the look and feel of **macOS**, right in your browser.

## Tech Stack

- ⚛️ **React 18** + **TypeScript** — UI framework
- ⚡ **Vite** — Lightning-fast build tool
- 🎨 **UnoCSS** — Atomic CSS engine
- 🐻 **Zustand** — State management
- 🎬 **Framer Motion** — Animations
- 🪟 **react-rnd** — Draggable & resizable windows
- 💬 **Gun.js** — P2P real-time IRC chat

## Features

- macOS-style desktop with draggable, resizable windows
- Dock with hover magnification animation
- Launchpad app grid launcher
- Spotlight search
- Multiple built-in apps (Safari, Terminal, VSCode, etc.)
- Dark mode support
- Desktop icons
- Mobile responsive with notification popup
